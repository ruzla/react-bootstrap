## Getting Started

### Clone down this repository:

```bash
git clone git@github.com:ruzla/weather-app.git
```

### Install dependencies

```bash
npm install
```

### Start up the application:

```bash
npm start
```

### Visit `localhost:8080` in your browser.

You should see the weather app.
